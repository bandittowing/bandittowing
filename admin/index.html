⚙️ One-time Cloudinary setup required — Before uploading photos, you need to create a free upload preset in Cloudinary:

1. Go to cloudinary.com → Settings → Upload
2. Scroll to Upload presets → click Add upload preset
3. Set Signing mode to Unsigned
4. Set Folder to bandit-gallery
5. Click Save — copy the Preset name it gives you
6. Paste that preset name into the box below, then click Save Preset.

You only do this once. After that, uploading photos works automatically forever.
